# Design Tokens – Concrete Values

## Colours
| Token | Hex / HSL | Usage |
|-------|-----------|-------|
| `--primary` | #3A7BD5 | Main UI background |
| `--accent` | #FFB71B | Buttons, highlights |
| `--text-primary` | #FFFFFF | Primary text |

## Typography
| Token | Font family | Weight | Size (rem) | Usage |
|-------|-------------|--------|------------|-------|
| `--font-base` | Inter, sans‑serif | 400 | 1rem (16px) | Body |
| `--font-header` | Inter, sans‑serif | 600 | 1.5rem (24px) | Headings |
| `--font-mono` | "Courier New", monospace | 400 | 0.875rem (14px) | Code blocks |

## Spacing
| Token | Value (px) | Usage |
|-------|-----------|-------|
| `--spacing-xs` | 4 | Tiny gaps |
| `--spacing-s` | 8 | Small gaps |
| `--spacing-m` | 16 | Standard padding |
| `--spacing-l` | 24 | Larger margins |
